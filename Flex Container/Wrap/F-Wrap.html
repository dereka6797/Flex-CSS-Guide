When it comes to flex wrap there isn't much more to it aside from what you'd think wrap means. Just like text this allows us to wrap items inside a container to essentially set up a new row. Now as you can see, I did leave some space to the right side of the colored squares just so show they will wrap when the width of the outer div is smaller than what's needed to fit said squares.

Inside your main div you want to display:flex & use the flex-wrap:wrap property. In the example I provided I use a width and height of 20rem for the items and a total width of 50rem for the container to show off a little space.

There is also wrap-reverse, which a lot like column and row reverse it simply puts the containers in reverse order. Below you will see the same squares we used above, but in reverse order.
